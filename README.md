I realized I'd been shipping lots of real projects, but lost touch with the fundamentals that got me there. So I decided to build something that solved a problem I dealt with, write each line by hand and really focus on understanding every library I was using. 

The idea for this project came from my ADHD, I would have a thought or note I wanted to write down and would either lose it in an apple note, or get paralyzed by a notion template. I wanted to build a tool where I can quickly type or paste in a note and have a background agent automatically organize it to where it belongs in my mental map. So that's what this is.
